# UOS API Tests

Zestaw automatycznych testów integracyjnych i walidacyjnych dla API systemu **UOS** (moduł Hotspot Vouchers).  
Technologia: **Python + Pytest**

---

## Uruchomienie testów

### Wymagania
- Python 3.10+
- pip
- Dostęp do środowiska UOS API (URL + token)
- Uzupełniony plik `uos.yaml` z konfiguracją testów

### Instalacja zależności
```bash
pip install -r requirements.txt
